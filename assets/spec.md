# spec.md

## 1. ソフトウェア概要
本ソフトウェアは、指定したパス内のファイルおよびディレクトリを一覧表示するコマンドラインツールである。  
標準的な ls コマンドの基本機能を簡易的に再現し、ディレクトリ内容を確認することを目的とする。

## 2. 入力

### 2.1 引数
- パス（省略可能）

### 2.2 入力例
my_ls
my_ls /home/user

## 3. 出力

### 3.1 出力内容
- 指定されたディレクトリ内のファイル名およびディレクトリ名を一覧表示する

### 3.2 出力例
file1.txt
file2.txt
dir1

## 4. 動作仕様
- 引数が指定されていない場合、カレントディレクトリを対象とする
- 指定されたパスが存在しない場合、エラーメッセージを出力する
- ファイルおよびディレクトリを区別せず表示する
- 表示順は特に保証しない（取得順）

## 5. 制約
- 再帰的なディレクトリ探索は行わない
- 詳細情報（サイズ・権限など）は表示しない
- オプション（-l, -a など）は実装しない
